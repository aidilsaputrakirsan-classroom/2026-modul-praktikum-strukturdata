# Nilai Final Praktikum Struktur Data (Kelas B)

> **Komposisi bobot:** Praktikum **30%**, Kuis **10%**, UTS **25%**, UAS **25%**, Kehadiran **10%**.

> **Kehadiran:** HADIR = 1, IZIN = 0.5, TANPA KETERANGAN = 0. Sel kosong pada minggu yang **sudah berjalan** dianggap alpa (0); sel kosong pada minggu yang **belum berjalan** di-skip. Skor = (sum / total minggu sudah berjalan) x 100.

> **Nilai Sementara:** dihitung dari komponen yang sudah ada (Praktikum + Kuis + UTS + Hadir), dinormalisasi ke skala 100 (asumsi UAS sebanding dengan rata-rata komponen yang ada). Akan dihitung ulang setelah UAS terisi.

> **Konversi huruf** (batas bawah inklusif): A >= 86, AB 76-86, B 66-76, BC 56-66, C 51-56, D 41-51, E < 41.

## Formula Perhitungan Final

```
Nilai Final = Praktikum x 0.30 + Kuis x 0.10 + UTS x 0.25 + UAS x 0.25 + Kehadiran x 0.10
```

Karena **UAS belum dilaksanakan**, kolom **Nilai Sementara** dihitung dengan normalisasi:

```
Nilai Sementara = (Praktikum x 0.30 + Kuis x 0.10 + UTS x 0.25 + Kehadiran x 0.10) / 0.75
```

Setelah UAS terisi, formula final akan menggunakan rumus pertama tanpa normalisasi.
