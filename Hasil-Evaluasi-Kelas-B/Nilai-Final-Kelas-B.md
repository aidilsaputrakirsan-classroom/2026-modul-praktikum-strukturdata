# Nilai Final Praktikum Struktur Data (Kelas B)

> **Komposisi bobot:** Praktikum **30%**, Kuis **10%**, UTS **25%**, UAS **25%**, Kehadiran **10%**.

> **Kehadiran:** HADIR = 1, IZIN = 0.5, TANPA KETERANGAN = 0. Sel kosong pada minggu yang **sudah berjalan** dianggap alpa (0); sel kosong pada minggu yang **belum berjalan** di-skip. Skor = (sum / total minggu sudah berjalan) x 100.

> **UTS & UAS:** ditampilkan nilai **final** (setelah remedial bila ada). Formula remedial: `nilai_final = MAX(asli, MIN(remedial, 65))`.

> **Catatan UAS:** kolom UAS = nilai **asli**. Remedial UAS (67 mhs) masih berjalan — deadline submit **Jumat, 19 Juni 2026 14.00 WITA**. Nilai mhs peserta remedial dapat naik (maks 65) dan **Nilai Akhir akan dihitung ulang** setelahnya.

> **Konversi huruf** (batas bawah inklusif): A >= 86, AB 76-86, B 66-76, BC 56-66, C 51-56, D 41-51, E < 41.

## Formula Perhitungan Final

```
Nilai Akhir = Praktikum x 0.30 + Kuis x 0.10 + UTS x 0.25 + UAS x 0.25 + Kehadiran x 0.10
```

UTS & UAS memakai nilai final pasca-remedial: `nilai_final = MAX(asli, MIN(remedial, 65))`.

Remedial UAS masih berjalan (deadline 19 Juni 2026). Setelah dinilai, isi `UAS_FINAL_OVERRIDE` di `compute_final_grades.py` lalu jalankan ulang script.
